## ✨ Features

- **📝 Multi‑step Form:** Uses `Ant Design Steps` to guide users through profile creation.
  - **Basic Information:** Name, subtitle, current work, fun fact, and more.
  - **Links:** Portfolio, blog and resume.
  - **Social Profiles:** Usernames for GitHub, LinkedIn, Twitter, etc. The generator builds complete URLs automatically.
  - **Skills & Technologies:** Pick your technologies from a list of XX+ icons, organised by category, with built‑in search.
  - **GitHub Stats:** Easily add visitor badges, statistics cards, and trophies.
- **🎨 Dark/Light Mode Support:** Choose your favourite theme – the setting is saved in your browser.
- **📱 100% Responsive Design:** The interface adapts perfectly to any device, from mobile to desktop.
- **👁️ Real‑time Preview:** See what the final README will look like as you fill in the data.
- **💾 Copy or Download:** With one click, copy the content to your clipboard or download the `README.md` file.

## 🛠️ Tech Stack

This project is built with the following technologies:

- **[React 18](https://reactjs.org/):** Main library for the user interface.
- **[TypeScript](https://www.typescriptlang.org/):** For robust, typed code.
- **[Vite](https://vitejs.dev/):** Blazing fast development tool.
- **[Ant Design](https://ant.design/):** Provides UI components (Steps, Layout, Cards, etc.) and theming system.
- **[Zustand](https://zustand-demo.pmnd.rs/):** Global state manager. Persists data between steps and stores theme preference.
- **[React Markdown](https://remarkjs.github.io/react-markdown/):** Renders the README preview.
- **[Axios](https://axios-http.com/):** For consuming the GitHub API.
